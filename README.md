## Get started

0. Prerequisite: [install Node.js](https://nodejs.org/en/download) (v18+ / LTS recommended).

1. Clone the repository
   `git clone https://github.com/AlleyCorpNord/bonfhir-hackaton.git`

2. Install dependencies
   `npm install`

3. Run the project
   `npm run dev`
